# Skill: HTML Email Rendering

## Objective
Render the structured knowledge base article into a professional email-safe HTML document using the supplied template.

## Responsibilities
- Use the attached template as the visual basis.
- Preserve an email-friendly layout with inline styling.
- Render Overview, Step 1, Step 2, Notes, and Summary clearly.
- Insert images or placeholders after relevant steps.
- Keep the output suitable for preview, copy, print, and download.
